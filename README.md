# Guess the Number Game (Python)

This is a simple, fun, and interactive number guessing game built using Python. In this game, the computer randomly selects a number between 1 and 100, and the player has to guess what that number is.

## 🚀 Features
* Random Number Generation: The computer selects a fresh, unpredictable number every time you start a new game.
* Smart Hints: If your guess is incorrect, the game guides you by telling you whether your guess was too high ("too big") or too low ("too small").
* Quit Option: Players can exit the game at any point by simply typing Quit.
  

## 🛠️ Technologies Used
* Python 3
* "random module" (Built-in Python module used for generating the target number)
